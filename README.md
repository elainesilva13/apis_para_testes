# Deploying the API to Vercel

To deploy the API to Vercel, follow these steps:

## Prerequisites
- Create an account on [Vercel](https://vercel.com)
- Install [Vercel CLI](https://vercel.com/docs/cli)

## Deploy Instructions
1. Open your terminal.
2. Clone the repository (if you haven’t already):
   ```bash
   git clone https://github.com/elainesilva13/apis_para_testes.git
   cd apis_para_testes
   ```
3. Run the following command to log in to your Vercel account:
   ```bash
   vercel login
   ```
4. To deploy your API, run:
   ```bash
   vercel
   ```
5. Follow the prompts to select your project and configure deployment options.

## Usage
- Once deployed, Vercel will provide a unique URL for your API.
- You can make requests to the API using this URL.
